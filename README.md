# ML_from_scratch
Implementations of machine learning algorithms from first principles. The goal of this repo is to move away from black-box libraries and understand the underlying optimization and math behind each model.

Contents:
    Regression: * LinearRegression.py: Ordinary Least Squares
        RegressionTree.py: Decision tree logic for continuous variables.
        SVR.py: Support Vector Regression.
    Classification:
        LogisticRegression.py: Sigmoid-based binary classification.
        svm.py: Support Vector Machine implementation.

Key Focus:
    No high-level frameworks (only NumPy/Math).
    Direct implementation of cost functions and gradients.
    Understanding algorithmic complexity and convergence.
